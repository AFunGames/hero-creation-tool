Welcome to the Hero Creation Tool for DnD 5e!

The Hero Creation Tool is a step-by-step process that makes building a character simple and streamlined. Changes are saved as you progress, but the character sheet is only updated when you review the data at the end.

Your Adventure Awaits!

The Hero Creation Tool currently only supports Dungeons & Dragons 5th Edition SRD from the Players Hand Book. You can add further content through the import function of the module or by creating custom modules on the fly in the creation tool.
